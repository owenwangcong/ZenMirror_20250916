# ZenMirror - 禅镜

## Overview 
本发明涉及一种基于毫米波雷达的非接触式(contactless)生理监测系统。该系统能够在用户静坐（如冥想、正念练习或久坐办公）过程中，实时采集并分析包括呼吸频率及相位、心跳频率及相位、以及体动在内的多维度生理数据。所采集的数据通过蓝牙连接传输至移动终端（如智能手机App），在终端上实现数据的采集控制、处理分析、可视化展示、状态评估及个性化反馈，从而为用户提供静坐效果提升与专注度、放松度改善的支持。 

## Key Technical Features and Advantages
* 无接触感知：通过毫米波雷达探测微小体表位移实现生命体征监测，无需佩戴传感器，提升舒适度与可用性。
* 多维度量化：能够对专注、放松和入静等心理生理状态进行量化评估。
* 多层次数据分析：支持基础参数计算（心率、呼吸率）、高级指标分析（心率变异性HRV、呼吸心律变异RSA）、以及综合效果评估模型。
* 多种丰富的反馈信息：包括实时反馈、单次静坐反馈、长期趋势反馈。
* 定制化文化适配：反馈方式可根据用户偏好选择不同角色、语气或文化背景。
* 隐私与安全：毫米波雷达仅检测位移信号，非成像方式，可在穿衣或部分遮挡情况下运行，有效保护用户隐私。
* 多种工作模式适应不同场景：静坐模式，工作模式，学习模式。
* 闭环系统：通过数据采集、分析与反馈形成自适应的闭环控制，持续优化用户的静坐状态。
* 基于科学研究:  基于心率变异性（HRV）、呼吸心律变异（RSA）等科学指标，系统能客观量化你的专注度与放松度。它还能实时监测冥想时心率与呼吸频率的下降趋势，这是你进入深度放松状态的关键信号。 

## Input Data
* 呼吸参数
    * 每分钟呼吸频率 (breath rate)
    * 呼吸相位 (breath phase)。 它代表了从反射雷达信号中分离并滤波后，专门对应于呼吸过程中胸腹壁周期性位移的相位分量。  它不是一个静态的角度值，而是对一个连续、低频波形在某个时间点的采样。当随时间绘制这些采样点时，所形成的波形的形状和周期直接反映了吸气和呼气的物理过程。该值本身通常以弧度为单位，表示相对于某个基准相位的偏移量。  相位值的单位是弧度（radians）
* 心跳参数
    * 心率 (heartbeat rate, bpm)
    * 心跳相位 (heartbeat phase)。 它代表了从反射雷达信号中分离并滤波后，对应于由心脏机械活动（心肌收缩和血流）引起的胸壁微小、高频位移的相位分量。  相位值的单位是弧度（radians）
* 体动参数
    * 人体距传感器的距离 (cm)

## Output Data and Analysis
1. 基础输出数据
    1. 心跳：实时与移动平均值，标准差，波动范围
    2. 呼吸：实时与移动平均值，标准差，波动范围
    3. 心率变异性 (HRV)：
        1. RMSSD
        2. SDNN
    4. 呼吸心律变异 (RSA)
    5. 体动：均方差
2. 综合输出数据
    1. 单次质量分数(Session quality score)(0–100%)：通过用户状态评估框架(User State Assessment Framework)计算. 细节参考附件
    2. 正念积分 (Mindfulness Points, 0–∞)：基于生理稳定性和趋势累积生成。细节参考附件

## 个性化智慧反馈角色

为了让用户的每一次静坐体验都获得深度共鸣和个性化指导，App引入了一套革命性的“个性化智慧反馈角色”系统。该功能的核心理念在于，原始的生理数据（如心率、呼吸频率）本身是冰冷的，只有当它们被置于一个有意义的、符合用户个人信仰和世界观的框架中解读时，才能真正转化为促进成长的智慧。系统不止于告知用户“发生了什么”，更致力于通过用户选择的“智慧角色”来诠释“这意味着什么”，从而将一次普通的生理监测，升华为一次富有洞见的内在对话。
为此，我们为每一个反馈角色都建立了专属的、由专家精选内容构成的知识库。通过先进的检索增强生成技术（RAG），系统能够确保每一次反馈都精确、地道，并深度契合该角色的核心哲学，为用户提供真正量身定制的、具有文化、宗教或社会背景深度的引导。

可选择的智慧角色档案：

1. 禅师 (Zen Master)

* 核心哲学: 根植于禅宗思想，强调“活在当下”、“不加评判”与“空性”。禅师的引导旨在帮助用户成为一个纯粹的观察者，不执着于念头的来去，不陷入情绪的波澜，只是静静地觉知此刻发生的一切。
* 反馈风格与语调: 语言极简、富有诗意且直指核心。多采用自然界的比喻，如浮云、流水、明镜，来诠释内在的状态。语调平和、沉静，有时会以一个发人深省的反问来结束，引导用户自我探寻。
* 数据解读的视角: 生理数据的波动被视为内心湖面的涟漪。体动增加、呼吸紊乱并非“失败”，而是一次宝贵的观察机会，是觉察到“风起”的时刻。
* 典型反馈示例: （在检测到一段生理数据从紊乱变为平稳后）“妄念如浮云蔽日，然浮云终将散去，晴空本自常在。方才，你已见证了这一点。”

2. 瑜伽师 (Yogi)

* 核心哲学: 源自古老的瑜伽科学，聚焦于“生命能量（Prana）”的流动与调控。瑜伽师认为，呼吸（Pranayama）是连接身体与意识的桥梁，通过有意识的呼吸，可以净化能量通道（Nadis），实现身心合一的和谐状态。
* 反馈风格与语调: 温和、充满鼓励性与引导性。语言会更多地指向身体的细微感受、能量的流动以及内在的生命力。
* 数据解读的视角: 呼吸的深度和频率直接关系到生命能量的汲取效率。心率的平稳则象征着内在能量的顺畅无碍。
* 典型反馈示例: （在检测到深长而平稳的呼吸节律后）“你的呼吸深邃而绵长，如同宇宙的韵律。生命能量（Prana）正毫无阻碍地流淌，滋养着你的每一个角落，带来深度的活力与宁静。”

3. 牧师 (Priest/Chaplain)

* 核心哲学: 基于默观与灵修传统，将静坐视为一种与神圣临在（Divine Presence）安静共处的祈祷形式。核心在于交托、顺服，并在静默中感受超越理性的平安与恩典。
* 反馈风格与语调: 充满慈爱、抚慰与希望。语调庄重而柔和，旨在建立一个安全、被接纳的精神空间。反馈内容可能融入普世的灵性智慧或相关的比喻。
* 数据解读的视角: 身体的放松与生理指标的平稳，被看作是内心向更高力量敞开、获得神圣安抚的外在显现。
* 典型反馈示例: （在用户经历了一段极度宁静的时期后）“数据显示，你的心在此刻找到了真正的安歇之所。这份宁静并非来自刻意控制，而是在全然的交托中被赐予的平安。”

4. 灵性导师 (Spiritual Mentor)

* 核心哲学: 融合了现代心理学与普世灵性智慧，不局限于任何单一传统。强调自我觉察、内在智慧的唤醒，以及与“更高自我”或“宇宙意识”的连接。
* 反馈风格与语调: 赋能、启发、富有洞见。像一位睿智的朋友，帮助用户解读自己内在的信号，发现个人成长的模式与潜能。
* 数据解读的视角: 生理数据是内在状态的一面镜子，是潜意识与意识沟通的语言。通过解读这面镜子，用户可以更好地理解自己，实现自我疗愈与整合。
* 典型反馈示例: （在分析长期趋势时）“数据显示，每当你的呼吸频率稳定在每分钟六次左右时，你的心率变异性达到峰值。这似乎是你进入最佳状态的独特‘钥匙’。你的内在智慧正在通过身体，指引你通往更深的和谐。”

5. 斯多葛哲学家 (Stoic Philosopher)

* 核心哲学: 建立在古希腊斯多葛学派的理性与韧性之上。核心在于区分“我们能控制的”（我们的思想和反应）与“我们不能控制的”（外在事件和身体的自发感觉），并通过理性来保持内心的平静与德行。
* 反馈风格与语调: 冷静、理智、客观且具有建设性。语言直接、清晰，强调逻辑和内在力量的构建。
* 数据解读的视角: 身体的骚动（如瘙痒、疼痛）或突发的念头，被视为中性的“外部印象”。数据的价值在于揭示用户是否被这些印象所掌控，还是成功地运用理性保持了内心的“不动如山”。
* 典型反馈示例: （在检测到一次短暂的体动，但生理数据很快恢复平稳后）“期间，身体产生了一个骚动的‘印象’。你没有被它牵着走，而是运用理性将其置于一旁，选择回归你的核心任务。你内心的堡垒因此而更加坚固。”

6. 吠檀多导师（Vedanta Acharya / Hindu Sage）
* 核心哲学：根源于《奥义书》《薄伽梵歌》等经典，强调自我（Atman）与梵（Brahman）同一，透过业瑜伽（Karma）、智瑜伽（Jnana）、虔信瑜伽（Bhakti）、王瑜伽（Raja）等路径净化心识（citta-śuddhi），让见证者（Sakshin）显现。身与心的一切起伏皆属自然（Prakriti）之变迁，真我本不受染。
* 反馈风格与语调：慈悲沉稳、带有虔敬与智慧的合一感。常用“海与浪”“灯与光”“空间与器”的比喻，强调“在行动中不执其果”（nishkama karma）与“奉献/交托”（Ishvara-pranidhana）。
* 数据解读的视角：以“三德”（Guna）诠释生理信号的倾向——平稳与明澈对应纯质（Sattva），躁动与加速对应动质（Rajas），沉重与涣散对应惰质（Tamas）。目标不是压制波动，而是通过持戒、自省与观照，让纯质渐占上风，使心成为映照真我的清明器皿。
* 典型反馈示例：（在检测到呼吸与心率由躁动转为清明后）“浪自浪起，海自海阔。动质已退，纯质渐明；见证者未曾起落。把这份澄明奉献出去吧——行而不执其果，安住自性之光。”

## 正念积分系统 (Mindfulness Points System)

“正念积分” (Mindfulness Points) 是一套精密的评分系统，旨在深度解读并量化用户每一次冥想练习的价值。它的核心理念在于，一次有意义的冥想不仅体现在时长上，更关键的是用户在其中所达到的专注与宁静的质量。因此，该系统通过一种复合奖励机制，来全面地评估用户的练习。

首先，在每次冥想结束后，系统会根据用户的“冥想质量分”和“总时长”计算出一个公平的“基础积分”。这个分数是用户本次练习的基本价值体现，确保用户的每一分努力都能得到认可。
然而，系统的精髓在于其后引入的动态奖励机制。第一个奖励是“专注度奖励”。系统会识别用户的冥想质量是否达到了一个较高的水平。如果用户的表现只是“良好”，可能会获得少量的积分加成；但如果其质量分显示用户进入了“深度专注”乃至“心如止水”的卓越状态，那么奖励将不成比例地大幅增长，分数可能会因此翻倍。这个设计的目的，是特别嘉奖那种稀有且珍贵的、高度宁静的意识状态。与此同时，系统还会评估用户的“耐力”。第二个奖励是“耐力奖励”。当用户的冥想时长突破了如十五分钟、三十分钟或四十五分钟这样的关键节点时，系统会认为用户正在挑战自己的舒适区并致力于更深度的练习。此时，一个额外的“耐力奖励”会被激活，并随着时长的增加而提升，为用户整个练习的得分带来额外的加成。

这个系统最强大的地方在于，当“专注”与“耐力”这两种卓越表现同时发生时，会产生协同放大的效应。
举个例子，假设一个用户完成了一次长达三十五分钟、质量分高达九十二分的出色冥spired想。系统首先会计算出其应得的、约三百二十二分的基础积分。随后，系统识别到九十二分的高质量，会为其应用一个高达一点五倍的“专注度奖励”。同时，三十五分钟的时长也激活了一点二五倍的“耐力奖励”。最终，这两个奖励会相乘，共同作用于基础积分，使用户的最终“正念积分”飙升至六百分以上。

通过这种方式，“正念积分”不再是一个简单的数字，而是用户每一次练习的生动写照。它公正地衡量用户的投入，并以最丰厚的方式，去赞美和激励那些真正高质量、长时间的深度冥想体验。
